# COLG论坛帖子搜索工具
一款轻量级的COLG论坛（DNF板块）帖子爬取与本地搜索工具，支持自定义爬取页数、多关键词模糊搜索、时间筛选、结果高亮等功能，数据本地存储，无需联网也能搜索。

![工具界面预览](https://img.shields.io/badge/界面-Tkinter-blue) ![打包格式](https://img.shields.io/badge/打包-EXE-green) ![Python版本](https://img.shields.io/badge/Python-3.8+-orange)

## ✨ 核心功能
1. **数据爬取**：自定义爬取COLG论坛DNF板块（forum-466）指定页数的帖子，自动去重
2. **本地搜索**：
   - 多关键词模糊匹配（如"剑魂技能"匹配"剑魂XXXX技能"）
   - 按发布时间倒序排序结果
   - 仅筛选2025-12-11之后发布的帖子
   - 时间高亮：2025-12-11前标红 / 之后标蓝
3. **便捷操作**：搜索结果标题可直接点击跳转至原帖
4. **数据管理**：支持清空本地数据库，数据文件存储在「我的文档/colg搜索工具」
5. **跨平台兼容**：数据库路径自动适配Windows「我的文档」目录

## 📥 快速使用（已打包EXE）
### 直接运行（无需Python环境）
1. 下载 `dist` 目录下的 `COLG帖子搜索工具_最终版.exe`
2. 双击运行，首次使用先点击「开始爬取」获取帖子数据
3. 输入关键词（如"剑魂技能"），点击「执行搜索」即可

### 数据存储位置
数据库文件自动生成在：
C:\Users\ 你的用户名 \Documents\colg 搜索工具 \colg_posts.db

📋 使用教程
1. 爬取数据
在「爬取页数」输入框填写需要爬取的页码（默认 50 页）
点击「开始爬取」，等待爬取完成（日志会显示每页爬取结果）
如需重新爬取，可点击「清空数据库」后再次爬取
2. 搜索帖子
输入关键词（支持多关键词，如 "剑魂 技能" 或 "剑魂技能"）
可选勾选「仅查看 2025-12-11 之后的帖子」进行时间筛选
点击「执行搜索」，结果按时间倒序展示：
🔴 红色：2025-12-11 前发布的帖子
🔵 蓝色：2025-12-11 及之后发布的帖子
⚫ 黑色：时间解析失败的帖子
点击标题可直接打开原帖
⚠️ 注意事项
爬取频率已设置为 0.8 秒 / 页，避免触发论坛反爬机制
若爬取失败，检查网络是否能访问 COLG 论坛（https://bbs.colg.cn）
若搜索无结果，确认：
已完成数据爬取
关键词输入正确（支持模糊匹配，但需包含所有关键词）
时间筛选条件是否过于严格
论坛页面结构若更新，需调整代码中crawl_page函数的 CSS 选择器
